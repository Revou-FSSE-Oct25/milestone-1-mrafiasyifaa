About Me

As a computer engineering student at Telkom University of Bandung, I have developed a strong foundation in hardware, network, computer architecture, and cloud computing. However, throughout my studies, I found myself curiously drawn to software development. This transition has led me to explore various programming languages and frameworks.

I've been involved in several projects, ranging from web application for goverment institutions to tackling confidential data of hospital management information systems. These experiences have honed my skills in backend development, database management, and API integration. I'm familiar with technologies such as Django, Flask, Node.js, and SQL databases. Join me in my journey as I set out to become a Fullstack Developer and contribute to the tech ecosystems.
