# Personal Branding - Portfolio Website

This repo is dedicated to house the source-code of my own personal portfolia and digital branding. The purpose of this
website is to share who am I and what I've been working on.

**[Kunjungi Live Demo di Sini](https://mrafiasyifaa.github.io/portfolio/)**

![Frontpage View](image-1.png)
---
## Website Main Features

This site is packed with following features:

* **Complete Navigation System**: Navigation menu to help you explore that sticks on top of the page
* **Structured Content**:
    * **Home**: Warm welcome to you and my profile.
    * **About**: Brief description of my background, skills, and interests.
    * **Projects**: Gallery of some of the things that I've built in Software Engineering field
    * **Contact**: Mail me an e-mail to keep in touch or just go to my social media at the bottom of the page.
* **Semantic Tags**: To keep my site on top of the SEO.
* **Music Player**: To keep you company while you are getting to know me.

## Technologies that I've Used to Build this Site
* **HTML5**: The backbone structure of the site.
* **CSS**: For styling and visual expressions.
* **JavaScript**: The brain of my site for interactivity
